Aplicativo Master Estoque

Este aplicativo de mercado foi criado para uma pequena empresa, inicialmente utilizando o python com Pandas e Tkinter, posteriormente foi necessária uma versão android (Java) para facilitar o processo. O aplicativo basicamente possui um menu com opções que realizam o cadastro de produtos, exibição de estoque, realização da venda, geração de relatórios em tempo real e o mercado que realiza as operações de venda em tempo real.
